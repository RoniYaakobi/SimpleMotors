# How to read the docs
## Matrix notation:
### When using matrix notation, matrices will be displayed as such:

```
    [ a , b ]
    [ c , d ]
```

### And vectors will be displayed as such:
```
    | x |
    | y |
```
### Multiplying vectors by matrices will be shown as such:
```
    | x | [ a , b ]
    | y | [ c , d ]
```
### When showing the output vector the following may also be shown:
```
    | x | [ a , b ] -> | x * a + y * b | == | ax + by |
    | y | [ c , d ] -> | x * c + y * d | == | cx + dy |
```
### For simplicity’s sake:
 - #### | | means vector
 - #### [ ] means matrix
 - #### | | [ ] means multiplying a vector by a matrix
 - #### -> means value by definition
 - #### == means equivalent
